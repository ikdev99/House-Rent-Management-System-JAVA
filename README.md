# 🏠 House Rent Management System

## 📌 Project Overview

The House Rent Management System is a Java-based desktop application designed to simplify the management of rental properties, tenants, and rent payments. The system provides a centralized platform for property owners and administrators to efficiently maintain tenant records, property information, and rental transactions.

This project was developed as part of an Object-Oriented Programming (OOP) course to demonstrate the practical implementation of software development principles and Java programming concepts.

---

# 🎯 Objectives

* Digitize rental property management processes.
* Maintain tenant and property information efficiently.
* Track rent payments and rental history.
* Reduce manual paperwork and record-keeping errors.
* Apply Object-Oriented Programming concepts in a real-world application.

4. Rent payments are recorded.
5. Reports and payment histories are generated.

# 🚀 Installation & Setup

## Clone Repository

```bash
git clone https://github.com/ikdev99/House-Rent-Management-System-JAVA.git
```

## Navigate to Project Directory

```bash
cd House-Rent-Management-System-JAVA
```

## Compile Project

```bash
javac Main.java
```

## Run Application

```bash
java Main
```
